[ 0x03. OPEN SOURCE PROJECTS ]

[ C | MEMORY MANAGEMENT | ACTIVE DEVELOPMENT ]
0x01. custom-malloc
Custom memory allocator implementation featuring segregated free lists, boundary tag coalescing, and first-fit allocation strategy. Built for understanding heap internals and heap exploitation primitives. Includes detailed documentation and test suite.
Tech Stack: C, x86-64 Assembly, GDB Scripts
[ PYTHON | EXPLOITATION TOOLKIT | STABLE ]
0x02. pwn-toolkit
Comprehensive collection of exploitation utilities for CTF pwn challenges. Features automated ROP chain generation, shellcode encoder/decoder, format string exploiter, heap visualization tools, and pwntools helper functions. Actively maintained with new modules added regularly.
Tech Stack: Python3, Pwntools, ROPgadget, Capstone
[ C++ | BINARY ANALYSIS | BETA ]
0x03. x64-debugger
Minimal x86-64 debugger built on Linux ptrace API. Supports software/hardware breakpoints, single-stepping, register inspection, memory dumping, and disassembly. Educational project for learning debugger internals and binary analysis fundamentals.
Tech Stack: C++17, ptrace, Capstone Disassembler
[ RUST | BINARY FORMAT | STABLE ]
0x04. elf-parser
ELF binary parser and analyzer written in Rust. Extracts and displays headers, sections, program segments, symbols, relocations, and dynamic information. Used for binary analysis, exploit development, and understanding executable file formats. Zero-copy parsing with nom parser combinators.
Tech Stack: Rust, nom Parser, ELF Specification
[ PYTHON | ROP AUTOMATION | EXPERIMENTAL ]
0x05. rop-compiler
Automated ROP chain compiler for x86-64 binaries. Analyzes available gadgets, constructs dependency graphs, and generates optimal ROP chains for specified operations including system calls, function invocations, and memory operations. Supports multiple ROP chain strategies and optimization passes.
Tech Stack: Python3, ROPgadget, Z3 SMT Solver, NetworkX
[ PYTHON | SHELLCODE | STABLE ]
0x06. shellcode-encoder
Polymorphic shellcode encoder for bypassing bad character restrictions and signature-based detection. Implements XOR, SUB, ROR encoding schemes with customizable decoders. Generates unique shellcode instances for each encoding operation. Essential tool for exploit development.
Tech Stack: Python3, Keystone Assembler, Shellcode Templates
